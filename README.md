# 面向生命科学的机器学习理论与方法课程
授课教师： 刘琦 教授 <br>
          同济大学 生命科学与技术学院 <br>
## 1. 课程简介
  机器学习是一门多学科交叉专业，涵盖概率论知识，统计学知识，近似理论知识和复杂算法知识，使用计算机作为工具并致力于真实实时的模拟人类学习方式， 并将现有内容进行知识结构划分来有效提高学习效率。 <br>
